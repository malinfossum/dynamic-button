# Dynamic Button

A small interactive web application where the user can dynamically generate buttons with custom text and colors.

---

## Features

- Dynamically creates buttons based on user input  
- Uses parameters and global state variables  
- Reads input values via `this.value`  
- Allows creating multiple buttons  
- Re-renders the view using a simple MVC-inspired structure  

---

## Focus

- Parameters  
- Variables (state management)  
- `this.value`  
- Functions returning HTML  
- DOM updates via re-rendering  

---
